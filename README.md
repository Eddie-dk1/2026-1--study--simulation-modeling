# Simulation Modeling Labs

Структура проекта приведена к формату, аналогичному примеру:

- `labs/lab04/project` — исходный код проекта лабораторной работы;
- `labs/lab04/report` — исходник отчёта;
- `labs/lab04/presentation` — исходник презентации;
- `labs/lab04/release` — файлы для релиза;
- `labs/lab04/sources` — архивируемые исходные материалы.
